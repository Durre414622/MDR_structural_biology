# Data Directory

This directory contains data files for structural analysis.

## Contents

- `5x1x.pdb` - NMR structure of Nudix hydrolase (PDB 5X1X)
  - Download from: https://files.rcsb.org/download/5x1x.pdb
  - Or run `src/structure_validation.py` which downloads it automatically

## Source

- PDB 5X1X: https://doi.org/10.2210/pdb5X1X/pdb
